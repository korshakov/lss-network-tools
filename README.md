# LSS NetInfo

Professional macOS network diagnostics toolkit.

Designed for IT engineers performing onsite network audits.

## Features

- Automatic dependency detection
- Interface discovery (WiFi / Ethernet / USB adapters)
- Network device discovery
- Gateway port scanning
- Firewall fingerprinting
- Rogue DHCP detection
- Network topology scanning
- Web admin interface detection
- Remote access detection
- Internet speed testing
- Quick audit mode
- Session export for documentation

## Requirements

macOS with:

- Homebrew
- nmap
- arp-scan
- speedtest-cli

The script can automatically install missing dependencies.
